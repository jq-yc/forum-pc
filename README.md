# forum-pc

## commitlint

| 类型     | 描述                                                                               |
| -------- | ---------------------------------------------------------------------------------- |
| build    |                                                                                    |
| ci       | 主要目的是修改项目继续集成流程(例如 Travis,Jenkins,GitLab CI,Circle)等的提交       |
| docs     | 文档更新                                                                           |
| feat     | 新特性                                                                             |
| fix      | 修复 bug                                                                           |
| perf     | 更改代码，以提高性能                                                               |
| refactor | 代码重构（重构，在不影响代码内部行为、功能下的代码修改）                           |
| revert   | 恢复上一次提交                                                                     |
| style    | 不影响程序逻辑的代码修改(修改空白字符，格式缩进，补全缺失的分号等，不改变代码逻辑) |
| test     | 测试用例新增，修改                                                                 |
| chore    | 不属于以上类型的其他类型                                                           |
